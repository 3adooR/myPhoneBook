== Описание ==
Простая телефонна книга со следующими возможностями:
* Регистрация
* Авторизация
* Возможность напомнить пароль
* Добавление записей в книгу
* Редактирование записей в телефонной книге
* AJAX-загрузка фотографий

= HTML + CSS (вёрстка) =
* Вёрстка построена на GRIDах.
* Адаптив.

== JS ==
* Используется как классический JavaScript-синтаксис, так и jQuery (в основном).
* Используются скрипты для проверки:
** E-mail адреса;
** Пароля;
** Телефонного номера;
* Переключение между разделами осуществляется без перезагрузки страниц (AJAX);
* Сессия автоматически продляется, чтобы не "обрывалась" авторизация;
* Используется библиотека FancyBox для вывода уведомлений и сообщений об ошибках;
* Используется библиотека TableSorter для сортировки строк в таблицах.

== PHP ==
ООП под PHP 7.2 + PDO.
Помимо описанных в описании (выше) функций есть следующее:
* Используется библиотека PHPMailer для отправки email-сообщений;
* Используется библиотека NumToText для преобразования цифр в строки ("1" в "один" и т.п.);

* Используется библиотека для простых строковых функий:
** Функция транслитирации строки (для преобразования названий файла из кириллических в латиницу);
** Преобразование стоки в телефонный номер (убирает всё ненужное);
** Автоматический вывод даты создания проекта для копирайта.

* Класс для работы с базами данных (DBClass), содержащий все основные функции по взаимодействию с БД:
** Подключение;
** Получение полей / строк;
** Проверка существования таблиц;
** Функии для обновления / удаления / добавления новых записей;
** Все функции PDO.

== DB ==
Используется база данных MySQL.
Подключение осуществляется через PDO.
